# Installation
using npm

```
npm install beem-components
```
using yarn

```
yarn add beem-components
```
React example
 ```javascript
import React from "react";

import { BmButton } from "beem-components";

function App() {
  return (
    <div>
      <BmButton variant="primary">Test</BmButton>
    </div>
  );
}

export default App;
```
